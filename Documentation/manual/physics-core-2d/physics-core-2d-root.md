## Unity's Physics Core 2D API

> [!IMPORTANT]
> Originally Unity called the Physics Core 2D API the low level physics 2d API when it was released in Unity 6.3.
> In Unity 6.5 the API was made as the new core physics 2D api for Unity and it was renamed from Low Level Physics 2D to Physics Core 2D API.
> They also updated the namespaces as well to Unity.U2D.Physics.


This page goes over the Unity's Physics Core 2D API and how it is used in the SF Metroidvania Toolkit.
